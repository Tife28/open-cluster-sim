# Mini Automotive Dashboard ECU

A simulated automotive instrument cluster built with Arduino and Tinkercad.

## Features

- ECU Boot Animation
- Digital Speedometer
- Fuel Gauge
- Engine Temperature
- Turn Indicators
- Hazard Lights
- Automatic Headlights
- Seatbelt Reminder
- Dashboard Warning System

## Hardware

- Arduino Uno
- LCD 16x2 (I2C - MCP23008)
- Potentiometers
- LEDs
- Push Buttons
- Piezo Buzzer
